# Legacy Archive Overview

This archive is intentionally small now. Most stale handovers, duplicated roadmaps, and dead bugfix notes were removed during cleanup.

The files that remain are the ones that still carry distinct value:
- `UNSORTED_AUDIT.md` records earlier archive triage decisions.
- `TEAM_FIELD_STRUCTURE_GUESS.md` is the largest surviving technical artifact from the old reverse-engineering notes.
- `verify.txt` preserves raw XOR verification snapshots.
- `breadcrumbs.csv` preserves collected binary-analysis offsets.

Everything else should be considered superseded by the maintained docs under `docs/` and the implementation under `app/`.
